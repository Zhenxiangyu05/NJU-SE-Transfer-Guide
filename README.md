# NJU-SE-Transfer-Guide

2026 南京大学软件工程最新转专业指南。

我是南京大学 2024 级技术科学试验班学生。大一分流时，我选择了数字经济专业，来到苏州校区学习。大二上学期，我萌生了转专业的想法，并于 **2025 年 11 月** 开始从零准备，最终于 **2026 年 7 月** 成功跨校区转入南京大学软件学院。

我并非二战考生，高中阶段也没有任何计算机基础，可以说是从零开始学习。加之身处苏州校区，跨校区修读专业课程十分不便，因此整个备考过程充满了挑战。正因如此，我深知转专业过程中信息获取困难、备考方向不明确所带来的迷茫与焦虑。

一路走来，我得到了许多老师、学长学姐以及同学们的帮助，也因此希望能够将这份善意继续传递下去。

创建这个仓库的初衷，是希望系统整理南京大学软件工程转专业相关信息，分享自己的备考经验和学习心得，为后来准备转专业的同学提供一份较为完整、可靠的参考。如果这份指南能够帮助你少走一些弯路，那便是它最大的意义。
